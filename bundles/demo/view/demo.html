Sujet 1 : Créer une mini API
Développer en PHP une mini API REST avec output des données en JSON.
1. Mettre en place de quoi pouvoir récupérer les données :
a. d'un utilisateur en fonction de son identifiant, les données étant son identifiant, son nom, son Email,
b. d’une chanson en fonction de son identifiant, les données étant son identifiant, son nom, sa durée.
2. Cette API doit aussi être capable de renvoyer les chansons favorites d'un utilisateur. Elle doit permettre de :
a. récupérer cette liste,
b. ajouter une chanson dans cette liste,
c. enlever une chanson de cette liste.
En développant cette API, vous devez garder en tête qu'elle peut être susceptible d'évoluer (nouveaux types d’output, nouveaux attributs dans les objets, nouveaux objets...).
Vous avez à votre disposition une base MySQL.
Exercice à réaliser sans frameworks ou librairies (jQuery, Zend, Symfony, ...)
